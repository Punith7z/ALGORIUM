# Algorium – MVJ E-Learning Platform

Algorium is a modern edtech platform developed to provide structured learning in high-demand technology domains such as Artificial Intelligence, Data Science, and Software Development. The platform is designed with a premium user interface and focuses on delivering an engaging and scalable learning experience.

---

## Overview

Algorium is built as a full-stack web application with a modular architecture. It integrates dynamic content rendering using EJS templates and follows a layout-based design system for maintainability and scalability.

The platform includes a homepage with premium UI sections, a course exploration system, and a video learning interface inspired by modern edtech products.

---

## Features

- Structured layout system using EJS and ejs-mate
- Premium homepage with modern UI design
- Course video player with playlist and navigation
- Resume playback and progress tracking (local storage)
- Notes and bookmark system with timestamp navigation
- Basic quiz integration for learning reinforcement
- Responsive design using Bootstrap
- Modular and reusable component structure (navbar, footer, sections)

---

## Tech Stack

### Frontend

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla)

### Backend

- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- ejs-mate (layout engine)

## Installation and Setup

1. Clone the repository - git clone https://github.com/punith7z/algorium.git
2. Navigate to the project directory - cd algorim
3. Install dependencies - npm install
4. Start the server - nodemon app.js
5. Open in browser - http://localhost:8080

---

## Usage

- Visit the homepage to explore the platform
- Navigate through different sections such as courses and learning modules
- Use the video player to access course content
- Track progress and add notes while learning

---

## Key Implementation Details

- Layout management is handled using `ejs-mate`
- All pages extend a common layout (`boilerplate.ejs`)
- Reusable UI components are stored in the `includes` directory
- Static assets are served using Express middleware
- Frontend interactions use vanilla JavaScript for performance and simplicity
